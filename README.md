# ph-porter

A ph-clint-based project.

## Getting started

```sh
pnpm install
pnpm dev
```

## Enabled features

- **Powerhouse**: off
- **Mastra agent**: on
- **Routine loop**: off

## Regenerate

Toggle features or update metadata in `.ph/ph-clint-cli/project-spec.json`,
then re-run `ph-clint clint-project-regen` to regenerate.
